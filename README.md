Objective
========= 

To provide simple self-contained examples of C++ concepts, both to help myself learn the language better, but also to help teach others.  The goal is to teach C++ 11, 14, 17 topics and identify them appropriately. 

Audience 
========

Users new to C++, new programmers, C++ programmers trying to learn new features in newer versions of C++.  

Requirements
============

* CMake Version 2.6 or higher 
* CMake version 3.8 for full C++17 compliance
* Each project will detail what level of C++ compliance is expected

Building
========

1. cd 01-helloworld <or other dir> 
2. cmake -H. -Bbuild 
3. cd build
4. make
5. ./01-helloworld <or other executable>
